# slack_bot_cleaner

This repo holds some utility code to delete messages from the slack bot from from a channel.

## Requirements

* Python 3.6
* [slack_cleaner2](https://pypi.org/project/slack-cleaner2/)

## Usage

```bash
python3 clean_slack.py <slack_token> <channel_name>
```
